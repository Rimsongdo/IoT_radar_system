OUR IoT RADAR

Our team has developed a detection system based on Internet of Things (IoT) technology to monitor and visualize environments in real-time. We used Arduino IDE framework to program the radar systems based on ESP8266 MCU. We have established bidirectional communication between the server and the ESP8266, and also between the server and the client. The Project Ressources can be found on Github through this following link. Click Here

The web interface has been implemented with HTML, CSS, JavaScript and Nodejs.

The P5.js library allowed us draw some interactive graphs!

OUR GOALS

Our primary objective is to develop and deliver an innovative, yet cost-effective, radar-based monitoring system with a web interface that caters to a diverse range of applications. We aim to provide a solution that is not only versatile but also intuitive and easy to use, enabling users from different fields such as security, home automation, and robotics to benefit from its capabilities.

By offering a comprehensive yet accessible toolset, we strive to empower individuals and organizations to efficiently monitor and manage their environments in real-time, enhancing safety, security, and productivity.

With our state-of-the-art technology, we seek to revolutionize the way monitoring tasks are carried out, providing an all-encompassing solution that adapts to the evolving needs of our users.

Key Features

Precise Detection: Our radar uses an ultrasonic sensor and a servo motor to scan its environment accurately and detect objects at different distances and angles.

Intuitive Web Interface: We have created a user-friendly web interface that allows users to visualize real-time radar detection data and adjust settings according to their needs.

Flexibility: Our system is designed to be modular and expandable, allowing users to add additional features or integrate the radar into other IoT projects.

TECHNOLOGIES

ESP8266: We use the ESP8266 microcontroller to control the radar and establish a Wi-Fi connection for communication with the web interface.

Ultrasonic Sensor: The ultrasonic sensor measures the distance between the radar and surrounding objects by emitting ultrasonic waves and measuring the time of return.

Servo Motor: The servo motor is used to rotate the ultrasonic sensor and scan the environment to detect objects in different directions.
